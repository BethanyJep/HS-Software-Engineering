# Career Paths in Tech 🌍

> A beginner-friendly guide to the many different careers you can build in software and technology.
> Made for **Grade 10 students at Alliance Girls High School** — but useful for anyone curious about tech.

---

## Why this folder exists

"Software engineering" is not one job — it is a whole world of jobs. Some people build the
websites you use, some protect them from hackers, some teach computers to think, and some
find hidden stories inside data. **You do not have to know which one is for you yet.**

This folder gives you one guide for each major tech field. Every guide is written the same way
so you can compare them easily:

- **What is it?** — a simple explanation, no jargon
- **What would I actually do?** — a day in the life
- **Is this me?** — signs this field might suit you
- **Skills to build** — what to learn, in order
- **Free resources** — courses, videos, and sites (mostly free)
- **Starter projects** — things you can build *this term*
- **Your first steps** — a checklist to begin today

## A note before you start 💛

> You do **not** need a fancy laptop, a lot of money, or to be a "maths genius" to start any of these.
> You need curiosity and the willingness to try, break things, and try again.
> Every expert in every field on this list started exactly where you are now.

Pick **one** guide that excites you and read it fully. Then try just the *first* starter project.
That single step is how careers begin.
